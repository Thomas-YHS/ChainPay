## 架构概览

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  前端 React │────▶│  Go 后端     │────▶│ PostgreSQL │
│  + wagmi    │     │  + Gin + Cron│     │            │
└─────────────┘     └──────┬───────┘     └─────────────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
     ┌─────────────┐ ┌──────────────┐ ┌──────────┐
     │ ChainPay    │ │  Li.Fi 合约  │ │ USDC    │
     │ Solidity    │ │  (Base)      │ │ (Base)  │
     └─────────────┘ └──────────────┘ └──────────┘
```

## 核心数据流

**手动发薪**：雇主前端 → Li.Fi SDK 生成 calldata → 前端调用 `executePayout` → 合约从雇主钱包拉 USDC → Li.Fi 路由到员工多链钱包

**定时发薪**：Go Cron → 执行者钱包调用 `executePayout` → 同上

**规则存储**：员工前端 → 直接写入 ChainPay 合约（`setRules`），不经过后端
