## Engineering 插件功能一览

| 技能                              | 触发命令                           | 用途                                      |
| --------------------------------- | ---------------------------------- | ----------------------------------------- |
| `engineering:documentation`     | `/engineering:documentation`     | 写技术文档（API文档、README、架构文档等） |
| `engineering:testing-strategy`  | `/engineering:testing-strategy`  | 设计测试策略和测试计划                    |
| `engineering:architecture`      | `/engineering:architecture`      | 创建或评估架构决策记录（ADR）             |
| `engineering:standup`           | `/engineering:standup`           | 从近期活动生成每日站会更新                |
| `engineering:system-design`     | `/engineering:system-design`     | 设计系统、服务和架构                      |
| `engineering:debug`             | `/engineering:debug`             | 结构化调试——复现、隔离、诊断、修复      |
| `engineering:deploy-checklist`  | `/engineering:deploy-checklist`  | 上线前验证清单                            |
| `engineering:tech-debt`         | `/engineering:tech-debt`         | 识别、分类、优先排序技术债务              |
| `engineering:incident-response` | `/engineering:incident-response` | 事故响应流程——分类、沟通、复盘          |
| `engineering:code-review`       | `/engineering:code-review`       | 代码审查——安全、性能、正确性            |
